## Key Focus Areas Demonstrated
- **Responsive & Adaptive Design:** Utilizing Flexbox and Grid for a seamless experience on mobile, tablet, and desktop.

## Project Structure and Contents
The portfolio is logically sectioned to present a full professional profile:

- **Hero Section:** Highlighting core professional role and expertise.  
- **Experience & Education:** Detailed history of professional roles and academic background.  
- **Skills:** Categorized display of technical and soft skills (e.g., Python, CRM, Sales Strategy).  
- **Projects:** Showcase of major projects demonstrating automation and analytical capabilities.  
- **Contact:** Functional contact form powered by Formspree (as seen in `index.html`).  

---

## Technologies Used

| Technology | Role in Project |
|------------|----------------|
| **HTML5**  | Semantic structure, SEO meta-tagging, and content organization |
| **CSS3**   | All custom styling, layout management (Flexbox/Grid), and animation effects |
| **JavaScript** | Minimal use for navigation (mobile menu toggle) and sticky navbar background |
| **Formspree** | Backend service used to handle contact form submissions |

## Installation and Local Setup
Since this project is built using basic HTML, CSS, and minimal vanilla JavaScript, no complex installation or dependencies are required.

# Clone the Repository
git clone https://github.com/dippalitofficial/dip-palit-portfolio

# Navigate to the Directory
cd dip-palit-portfolio

# Launch
 Open the index.html file in your web browser


# Project Files
**File	Description**

index.html	The main file containing all the website content, structure, and embedded scripts
style.css	Comprehensive CSS stylesheet defining all visual aspects, responsive breakpoints, and custom effects (e.g., Glassmorphism styles)
robots.txt	Standard file providing directives for search engine crawlers
sitemap.xml	XML sitemap to help search engines efficiently index the portfolio pages
